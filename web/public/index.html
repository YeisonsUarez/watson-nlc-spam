SMS Spam / Ham Classifier

This example Watson Natural Language Classifier was training using ~5000 examples provided by the SMS Spam Collection.

The SMS Spam Collection v.1 is a public set of SMS labeled messages that have been collected for mobile phone spam research. It has one collection composed by 5,574 English, real and non-enconded messages, tagged according being legitimate (ham) or spam. More information can be found here.
